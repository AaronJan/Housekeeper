# Housekeeper - Laravel

**`2.0-dev` is here!** More docs and tests are on the way!

For version `0.9.x`, here are the [Documents](https://aaronjan.github.io/Housekeeper/0.9.x/).


## Introduction

Powerful `Repository-Pattern` implementation for Laravel `(>= 5.0)` and PHP `(>= 5.5)`, it's really cool and really simple to use.
